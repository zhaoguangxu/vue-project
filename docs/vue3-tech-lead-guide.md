## 技术选型与架构设计

### 核心框架

- Vue 3 作为核心框架
- 采用 Composition API 作为主要开发范式

### 配套技术栈

- **构建工具**：Vite（推荐）或 Webpack
- **状态管理**：Pinia（Vue 3官方推荐）
- **UI组件库**：Element Plus、Ant Design Vue等
- **TypeScript**：提供类型支持
- **路由**：Vue Router 4
- **测试框架**：Vitest + Vue Test Utils

- 使用 Conventional Commits

### 项目目录结构规范

```
src/
├── assets/        # 静态资源
├── components/    # 公共组件
├── composables/   # 组合式函数
├── layouts/       # 布局组件
├── router/        # 路由配置
├── stores/        # 状态管理
├── types/         # 类型定义
├── utils/         # 工具函数
└── views/         # 页面组件
```
